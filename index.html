Armadillo

Micro Framework hecho en python para ciencia de datos
Descargalo aquí →

¿Que hace ?

Armadillo es una implementación de código abierto construido para aquellas personas que están iniciando en el mundo de la ciencia de datos en el lenguaje python, con métodos rápidos para minar y modelar datos escrita por Leonardo Patiño Rodriguez.

Compartir

¿Como cargo el código ?

1. Abra el CMD y cree una carpeta nueva

mkdir armadillo && cd armadillo

2. Crear un entorno por separado para que corra armadillo.

conda create -n armadillo python=3.6

3. Ver entornos, garantice que el entorno se creo de manera correcta.

conda info --envs

4. Active el entorno

conda activate preventivo

5. navegue hasta la carpeta que creó en el paso 1 e instale las dependencias de armadillo.

pip install -r requirements.txt


¿Como uso el código ?

1. Arquitectura.

              logs
              api.py
              datasets.py

db: Se encuentran las clases y los métodos para acceder a los diferentes motores de datos.

clase: Se encuentra la familia de clases para hacer ciencia de datos, como limpieza de datos, análisis, modelos etc.

conexion: String de conexión para cada motor de datos en formato json

conf: Clases adicionales de configuración del micro framework

dataframe: En esta carpeta se encuentra los dataframes que se generan desde los metodos de armadillo.

ficheros: En esta carpeta se encuentran todos aquellos ficheros de entrada.

logs: Logs de error, info, debug, Armadillo deja un log por cada proceso por lo que se facilita bastante a la hora de necesitar un historial u error.

api.py: Si quieres transformar armadillo a un web service.

datasets.py: Biblioteca de datasets para probar.

info.txt: Información del micro framework.

main.py: Este es el archivo princial en donde los usuarios manipularian las diferentes clases y métodos que ofrece armadillo.
